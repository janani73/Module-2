### AIM  
To write a Python program that reads two numbers and calculates the power of the first number raised to the second using the pow() function.

### ALGORITHM
```
1.Start the program.
2.Read the base value x from the user.
3.Read the exponent value y from the user.
4.Compute the power using z = pow(x, y).
5.Display the result.
6.End the program.
```
### PROGRAM
```
x=int(input())
y=int(input())
z=pow(x,y)
print("Power of the given number is:",z)
```
